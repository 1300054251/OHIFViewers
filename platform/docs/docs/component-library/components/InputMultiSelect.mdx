# Input Multi Select

## Import

```javascript
import { InputMultiSelect } from '@ohif/ui';
```

## Basic usage

```jsx live
{
  () => {
    const [values, setValues] = useState([]);
    return (
      <div className="flex flex-col flex-1 p-4 items-center">
        <div className="w-56">
          <InputMultiSelect
            label="Modality"
            options={[
              { value: 'SEG', label: 'SEG' },
              { value: 'CT', label: 'CT' },
              { value: 'MR', label: 'MR' },
              { value: 'SR', label: 'SR' },
            ]}
            value={values}
            onChange={values => {
              setValues(values);
            }}
          />
        </div>
      </div>
    );
  };
}
```
